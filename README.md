## About The Project

Single-page application similar to Tw*tter hooked up to an existing API

### Built With

* Vanilla Javascript

## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:kerisic/chitter.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
